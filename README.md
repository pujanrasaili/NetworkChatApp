# NetworkChatApp 💬

A multi-client real-time chat application built with Java.

## Features
- Multiple clients connect simultaneously
- Real-time message broadcasting to all users
- Username support
- Built with Java Sockets and Threads

## How to Run
1. Run `ChatServer.java` first
2. Run `ChatClient.java` (open multiple instances to test)
3. Type messages and chat!

## Tech Stack
- Java Sockets (ServerSocket, Socket)
- Java Threads (Runnable)
- Java I/O Streams (BufferedReader, PrintWriter)
